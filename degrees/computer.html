What is Computer Science
If you’re interested in creating your own software, no matter if it’s a game, website, quality of life app, computer science is a great choice.
Software Engineering VS Computer Science
• Software Engineering is a subset of Computer Science, and only deals with the practical elements designing software. However, a Computer Science degree is heavily advised as even if you’re only interested in the applications of coding, Computer Science covers much more background knowledge such as complex mathematics that allows you to create software at a higher level.

App developer
App developer follow project specifications and create software that are used by a specific target audience, or the public. This is usually in the form of an application or app on desktop
Web Developer
Web developers are developers that create websites or web applications for the world wide web. Within web development, there are front end developers who are specialized in creating good user interface (actions that users can see) and back-end developers who are specialized in dealing with all the interactions with other websites, databases, APIs… basically actions that users will not interact with.
AI Engineer
AI engineers create Artificial Intelligence through training their code with big data conducted through many algorithms. Many businesses employ AI engineers to use artificial intelligence to increase the overall efficiency of their business, by first identifying opportunities for AI deployment, then train, test and refine machine learning models to improve the specific task.
Computer Scientist
Computer Scientists, unlike developers, do not create new software. They usually concentrate and research on abstract data structures and algorithms, and research and develop new methods to optimize these methods. They rarely write code and only think about the inner workings of a system.
