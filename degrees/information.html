What is Information Engineering?
If you’re fascinated by big data, information engineering is worth considering. Information engineers design, create and update information systems, such as servers.
Software Engineering VS Information Engineering
Software Engineers mainly do programming while information engineers do not program, but mainly build systems such as servers to store data.

Network engineer
Network Engineers’ main objective is to design, develop and implement all the communication network and testing systems for its performance. They establish network systems to improve network efficiency, monitor network traffic for packet losses (data loss), then troubleshoot the problem with a devised plan. Many also configure both the hardware and software of the organization which includes servers, switches, routers, and other hardware.
